# Space Ship Runner

Фреймворк а-ля спринг ... по мотивам доклада Евгения Борисова [Spring-построитель](https://www.youtube.com/watch?v=rd6wxPzXQvo) 

#### Используемые технологии
* Java 11
* Maven
* Reflections
* Lombok
* SOLID

#### Сборка и запуск
```
mvn clean package
java -jar target/space-ship-runner-1.0-SNAPSHOT.jar
```
